anndata_updater
===========================================

A simple tool to update AnnData objects for compatibility with other toolkits
and ecosystems; basic command-line usage is as follows

::

    python3 -m anndata_updater -i input.h5ad [-o output.h5ad]

Installation
------------

anndata_updater is available on GitHub at
https://github.com/mojaveazure/anndata_updater and can be installed with pip

::

    pip install git+https://github.com/mojaveazure/anndata_updater
